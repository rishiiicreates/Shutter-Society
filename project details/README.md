## 🎯 One-Line Vision

Build a website that **feels like a movie trailer** — immersive, dark, cinematic, and unforgettable.

## 🧠 Core Emotion Target

The first 5 seconds should make the visitor say:

```
"Damn… this is insane."
```

## 🏗️ Project Type

- **Framework:** Next.js 14 (App Router)
- **Style:** Dark, luxury, brutalist-cinematic
- **Animation Engine:** GSAP + Framer Motion
- **3D Layer:** React Three Fiber / Spline
- **Scroll:** Lenis (smooth scroll)

## 📌 Quick Start

```bash
# Clone and install
git clone https://github.com/your-org/content-group-website
cd content-group-website
npm install

# Run dev server
npm run dev

# Build
npm run build
```
